Compile with g++ queens.cpp -o queens
Run with: queens M N B W tmax fileName extra

M 		- rows
N 		- columns
B 		- # of black queens
W 		- # of white queens
tmax 		- time in seconds
fileName	- string for file name of solution "solution.txt"
Type “extra” to turn on random restarts. If not, type another string

Initial state file will also be created.

If you received Segmentation Fault 11, then you didn’t type in everything correctly
